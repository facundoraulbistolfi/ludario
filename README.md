# ♟️ tabletop-helper

> Herramientas web para juegos de mesa, cartas, sudokus y otras yerbas.

**🌐 [facundoraulbistolfi.github.io/tabletop-helper](https://facundoraulbistolfi.github.io/tabletop-helper/)**

---

## Herramientas

| Herramienta | Descripción |
|---|---|
| [Sudoku Killer](https://facundoraulbistolfi.github.io/tabletop-helper/tools/sudoku-killer/) | Solver de cages con filtros de dígitos, restricciones entre pares y acceso rápido a PDFs de KrazyDad |
| [Anotador de Chinchón](https://facundoraulbistolfi.github.io/tabletop-helper/tools/chinchon/) | Configura jugadores, suma rondas, marca chinchón o −10, y guarda/carga el estado de la partida |

## Deploy

El sitio se despliega automáticamente en **GitHub Pages** cada vez que se mergea a `main`, via GitHub Actions (`.github/workflows/deploy.yml`).
